# The Urban Barber - Website

Een responsive website voor **The Urban Barber**, gemaakt voor de opdrachten Klantwens & backlog, Functioneel ontwerp, GitHub & website-template en Website bouwen.

## Live website

[Open de live website](https://getrektbynoob15.github.io/studio-noord-website-template/)

## Klantwens

The Urban Barber wil een strak, modern en een beetje brutaal online visitekaartje. De site moet de ontspannen sfeer van de barbershop laten voelen en bezoekers snel helpen met diensten, prijzen, openingstijden, contact en het aanvragen van een afspraak.

### Must have

- Contactgegevens en openingstijden
- Diensten en prijzen
- Foto's van de zaak en het team
- Responsive ontwerp voor mobiel, tablet en desktop

### Should/Could have

- Online afspraak aanvragen
- Meet the Barbers-pagina met humor
- Link naar een playlist

### Out of scope

- Pop-ups, nieuwsbrief en chatbot
- Webshop en actiepagina

## Functionaliteiten

- Responsive homepage met navigatie, header en footer
- Diensten- en prijzenoverzicht
- Openingstijden en contactgegevens
- Extra pagina: Meet the Barbers
- Afspraakformulier met bevestiging
- Mobiele navigatie
- Automatische publicatie via GitHub Pages

## Technieken

- HTML5
- CSS3 en media queries
- JavaScript
- Git en GitHub

## Git-workflow

Werk niet direct op `main`. Maak voor iedere taak een eigen branch, bijvoorbeeld `feature/prijzen`, `feature/barbers` of `feature/responsive`. Open daarna een pull request en laat een teamlid de wijziging controleren voordat deze naar `main` gaat.
