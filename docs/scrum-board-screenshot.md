# Screenshot Scrum-bord

Plaats hier het bewijs van jullie samenwerking:

1. Open het Scrum-bord in GitHub Projects.
2. Zorg dat de taken verdeeld en bijgewerkt zijn.
3. Maak een screenshot waarop de kolommen en taken zichtbaar zijn.
4. Sla de afbeelding in deze map op als `scrum-board.png`.
5. Commit de screenshot via een eigen branch en pull request.
